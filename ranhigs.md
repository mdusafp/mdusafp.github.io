# Приложение для СЗИУ РАНХиГС

### Зачем?
- Только в yandex word stat слово ранхигс встречается в свыше 400 тысячах запросов в месяц
- На 2016-ый год число обучающихся по всем направлениям в академии насчитывало 200 тысяч человек
- Существует 200 программ дополнительного образования и 75 основных образовательных программ
- Текущий сайт состоит из огромного функционала, который не так нужен для поступающего
- Абитуриенту трудно определиться с дальнейшим будущим
- Вуз теряет абитуриентов из-за сложного процесса подачи документов
- Повышение числа абитуриентов

### Что необходимо сделать?
##### С точки зрения вуза:
Нужно увеличить ковнверсию абитуриентов.
##### С точки зрения разработчика:
Создание сервиса для привлечения абитуриентов

## Декомпозиция задач:
##### Для вуза
Основная проблема, которую решает это приложение - 
Большая часть абитуриентов теряется из-за того, что непонятно какой перечень документов нужен. Особенно ярко это проявляется, когда школьник ещё не определился со своей будущей стезей. 
Следовательно, необходимо сопроводить потенциального абитуриента до подачи документов в вуз.
Помочь ему с определением направления в зависимости от его склонностей и предоставить варианты куда он может поступить, показать преимущества и недостатки, а так же необходимый перечень документов для поступления на эти направления.

##### Для разработчкиов
Нужно написать SPA, позволяющий динамически подстраиваться под пользователя в зависимости от его предпочтений. 
Функционал, который предоставят юзеру - это форма, которая будет при помощи AJAX динамически менять контент на странице, когда пользователь будет отвечать как-то на поставленные вопросы.
Должны быть заготовлены различные тексты сопровождающие пользователя при выборе направления, вопросы и проанализированы различные события для построения серверной логики.
В качестве подбора решения будет список подходящих направлений по соответствующим ключам, хранимым в ответах на вопросы. Вывод после выполнения алгоритма должен содержать 3-5 наиболее благоприятных направления исходя от подобранного коэффициента схожести с интересов абитуриента и подразумеваемых интересов человека, учащегося на данном направлении.
Абитуриент во время прохождения этой анкеты оставляет данные о себе и заодно свои интересы, которые можно будет в дальнейшем использовать для анализа или холодных звонков.

### Что предстоит обсудить?
- Я считаю, что делать SPA анкету в качестве приложения на смартфон - затратно и не логично, но у нас в команде есть мобильные разработчики, которых можно применить получше.
- Так что стоит в первую очередь обсудить, что делать под мобильные платформы.
- Помимо SPA хотелось бы иметь написанные скрипты для рассылки на почты абитуриентов с полезной информацией и/или боты для рассылки в  telegram/whatsapp/viber/skype.
- Должен быть сервер, который будет поддерживать CRUD и уметь выбирать направления для будущих студентов СЗИУ РАНХиГС.
